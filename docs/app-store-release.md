## 商店文案草案

### 副标题

图片转拼豆色号图纸

### 推广文本

在手机上完成裁剪、像素化、色号匹配、手工改色和备豆统计，导出清晰的 PNG/PDF 拼豆图纸。

### 描述

拼豆图纸是一款面向拼豆创作者的本地图片转图纸工具。

- 从相册选择图片并裁剪
- 自定义成品宽高和豆子数量
- 使用 MARD 221 色卡，或导入店铺 CSV 色卡
- 控制色数与抖动强度
- 单格改色、框选填色、撤销与重做
- 自动统计每个色号的用量
- 本地保存和恢复作品
- 导出带网格、坐标、色号及图例的 PNG/PDF

图片转换和作品保存均在设备上完成，无需注册账号。

### 关键词

## 提交前清单

- 安装最新版 Xcode，并在 Xcode 登录 Apple Developer 账号
- 在 Apple Developer 注册与工程一致的显式 Bundle ID
- 在 Xcode 选择开发团队并开启自动签名
- 真机测试图片选择、离线启动、本地保存、PNG/PDF 分享

- 在 App Store Connect 创建应用，填写名称、分类、年龄分级和隐私信息
